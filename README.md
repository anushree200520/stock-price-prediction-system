
📈 Stock Price Prediction System using Machine Learning and Django


Overview

The Stock Price Prediction System is a web application developed using Python, Django, and Machine Learning. It predicts stock prices based on user inputs and displays historical stock trends through interactive visualizations. The project demonstrates the integration of machine learning with web development.




 Features

User Registration and Login
Predict stock prices using Machine Learning
View historical stock data
Interactive charts and visualizations
User-friendly web interface
Supports multiple company stock predictions

⚙️ Installation

1. Clone the repository:
git clone https://github.com/anushree200520/stock-price-prediction-system.git

2. Install dependencies:
pip install -r requirements.txt

3. Run the server:
python manage.py runserver

4. Open in your browser:
http://127.0.0.1:8000/
